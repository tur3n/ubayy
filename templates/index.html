This is a python web app written in django. The app has simple authentication system where entering email will automatically authorize existing user or create new user if none found. Users can place bid, update their bid, check other bids on a product. Users can create their own product listing with minimum bidding price and expiry datetime. Bid on a item has to be greater than the or equal to the minimum bid and has to be greater than the last bid.
